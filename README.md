## Đề tài: GroFast - Ứng dụng di động mua sắm tạp hóa theo yêu cầu

**Mô tả:** Xây dựng chuỗi ứng dụng phục vụ mục đích mua sắm tạp hóa theo yêu cầu tại thị trường Hà Nội, Việt Nam
